# Proyecto: Portafolio de Pablo Elias

Este archivo contiene el contexto y las directrices para el desarrollo y mantenimiento del portafolio personal.

## 🚀 Tecnologías
- **HTML5**
- **Tailwind CSS** (vía CDN)
- **Lucide Icons** (Iconografía)
- **Google Fonts**: Montserrat (Fuentes)

## 📁 Estructura del Proyecto
- `index.html`: Archivo principal con la estructura y estilos.
- `assets/`: Carpeta que contiene las imágenes del portafolio.
  - `ágora/`: Iconos y recursos del proyecto Ágora.
  - `lemonth.png`, `Fujita-VerticalView.png`: Imágenes de proyectos.
  - `Foto perfil.png`, `hero.png`: Fotos personales.

## 🎨 Estilo Visual
- **Fondo:** `#0A0A0A` (Negro profundo)
- **Acento:** `#E50914` (Rojo vibrante)
- **Texto:** Blanco y gris claro para jerarquía visual.

## 🛠 Instrucciones para Iterar
1. Siempre usa **Tailwind CSS** para nuevos estilos.
2. Mantén la estética minimalista y profesional del diseño original.
3. Asegúrate de que cualquier nueva sección sea responsiva.
4. Las imágenes deben ir siempre en la carpeta `assets/`.
